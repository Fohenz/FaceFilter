## Using Make  
```bash  
make download  
make  
make run  
make clean  
```  



## Without Make

### Compile  
```bash  
g++ face_landmark_ex.cpp -o test -std=c++11 -ldlib -lcblas -llapack
```  
### Run  
```bash  
./test shape_predictor_68_face_landmarks.dat face.jpg  
```

### pre-learned data file
```bash  
wget http://dlib.net/files/shape_predictor_68_face_landmarks.dat.bz2
```
